# 目录插件

启用插件后，会在顶栏右侧生成一个插件图标

左键点击插件图标，在当前文档下插入目录列表

右键点击插件图标，显示更多操作

`CTRL + ALT + I`插入目录

`CTRL + ALT + O`插入子文档目录及大纲

`CTRL + ALT + P`插入当前文档大纲

`CTRL + ALT + N`插入笔记本目录
